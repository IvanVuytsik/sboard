# SBoard — Рендеринг Pixi.js через Skia с экспортом в PDF

Приложение на TypeScript, которое рендерит граф сцены Pixi.js через пользовательскую обёртку Skia (CanvasKit) с функционалом экспорта в PDF.

## Возможности

- **Двойной рендеринг на canvas**: Сравнение рендеринга Pixi.js (эталон) и Skia (пользовательская обёртка)
- **Поддержка трансформаций**: Полная поддержка translate, rotate и scale во вложенных DisplayObjects
- **Рендеринг графики**: Объекты PIXI.Graphics рендерятся как векторные пути в Skia
- **Поддержка спрайтов**: Объекты PIXI.Sprite рендерятся как растровые изображения в Skia
- **События указателя**: Интерактивные события pointerdown/pointerup на обоих canvas с проверкой попаданий
- **Экспорт в PDF**: Экспорт отрендеренных сцен в векторный PDF через PDF-бэкенд Skia
- **Генерация случайных фигур**: Генерация случайных фигур для тестирования
- **Сброс сцены**: Сброс сцены в начальное состояние

## Требования

- Node.js v26 или выше
- npm или yarn

## Технические детали

### Конфигурация Pixi.js
- Версия: pixi.js@7.2.4
- Режим canvas: forceCanvas=true
- Размер canvas: 400x300 (компактный макет)

### Skia CanvasKit
- Пользовательская сборка с поддержкой PDF
- WASM загружается из локальной директории public/
- Поддержка векторного рендеринга и экспорта PDF

### Система трансформаций
- Матричная композиция трансформаций
- Поддержка вложенных трансформаций контейнеров
- Совместима с порядком трансформаций Pixi (position → rotation → scale)

### Экспорт в PDF
- Векторные пути для объектов Graphics
- Растровая отрисовка для объектов Sprite
- Скачивается как `scene-export.pdf`

## Использование

### Взаимодействие с canvas
- Кликните на объекты в любом canvas для запуска событий указателя
- События логируются в консоль
- Визуальная обратная связь (изменение alpha) на Pixi canvas

### Элементы управления
- **Generate Random Shape**: Добавляет случайную фигуру (прямоугольник, круг или треугольник) на сцену
- **Export PDF**: Экспортирует текущую сцену в PDF файл
- **Reset Scene**: Сбрасывает сцену в начальное демо-состояние

### Панель статуса
Панель статуса показывает текущую операцию и любые ошибки.
